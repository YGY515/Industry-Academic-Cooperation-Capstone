# Industry-Academic-Cooperation-Capstone
### 졸업 작품으로 진행한 Unity 3D 콘텐츠 개발 프로젝트
<b>프로젝트 기간: 2023. 04 ~ 2023. 11</b> <br><br>
팀장을 맡아 개발팀 5명, 디자인팀 4명 총 9명을 이끌고 졸업 전시회에서 장려상을 수상하였습니다.
<br>
한이음 ICT멘토링과 연계하여 한국공학대 디자인공학부 팀원들과 진행하였습니다. 

Unity 3D와 Plastic SCM을 활용하였고, Notion과 Figma를 통해 일정 관리 및 프로젝트를 관리했습니다. <br>
인게임 모델링과 UI 아이콘은 한국공학대 디자인공학부 팀원들이 담당한 에셋입니다. <br>

팀장으로서 디스코드와 노션을 통해 프로젝트 일정을 관리하고, 회의록을 기록하며 개발 진도를 공유했습니다.<br>
개발 팀원들의 역할을 분배하고 UI를 담당했으며, 작업 효율을 위해 Unity Verson Control을 구축하여 프로젝트를 관리했습니다.<br>
캡스톤 강의 발표와 전시회 발표를 담당하여 제작 결과물을 교수님들에게 공유했고, 최종적으로 졸업 작품 전시회에서 장려상을 수여받았습니다. <br>
